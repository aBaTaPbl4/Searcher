Используемые дополнительные компоненты(см папку Libs):
-spring.net
-nunit
-rhino mocks
-log4net

Реализованная архитектура:
MVVM + SOA

Реализованные фичи:
-Система плагинов
-Динамическая подгрузка плагинов в режиме реального времени
-4 модуля: 
	Поиск по атрибутам файла 
	Поиск по текстовым файлам
	Поиск по сборкам .NET
	Поиск по xml файлам
-Три режима работы программы: 
	С одним фоновым потоком 
	С кол-вом потоков, определяемым пользователем
	С кол-вом потоков, определяемым системой
-Возможность работать с найденным множеством файлов:
	Возможно удалять один, несколько или все файлы
	Возможно выполнить переход к выбранному файлу
